# Job Application Tracker

A professional and responsive **Job Application Tracker** built using **HTML, CSS, and Vanilla JavaScript**.  
This application helps users efficiently manage and track their job or internship applications in a single place.

🔗 **Live Demo:**  
https://yashwanthr12.github.io/job-application-tracker/

---

## Overview

Managing multiple job applications can be challenging.  
This project provides a simple solution to track application details such as company name, role, application date, status, and notes, with persistent storage using the browser’s localStorage.

The project focuses on **core frontend fundamentals** without using any frameworks.

---

## Features

- Create new job or internship applications  
- Edit existing applications  
- Delete applications  
- Track application status (Applied, Interview, Offer, Rejected)  
- Search applications by company name or role  
- Filter applications by status  
- Persistent data storage using localStorage  
- Responsive and user-friendly interface  

---

## Tech Stack

- **HTML5** – Semantic markup and structure  
- **CSS3** – Styling and responsive layout  
- **JavaScript (ES6)** – Application logic and state management  
- **localStorage** – Client-side data persistence  

---

## Application Architecture

- State-driven UI rendering  
- Single source of truth using JavaScript arrays  
- Event delegation for edit and delete actions  
- Reusable functions for rendering and storage  

This structure ensures clean, maintainable, and scalable frontend code.

1. Clone the repository  
2. Open `index.html` in any modern browser  
3. Start adding and managing job applications  

No additional setup or dependencies are required.
